Basic Html Tags

  1. Heading tag-Used to separate headings and subheadings on a webpage
  2. Image tag-Used to add an image inside a webpage
  3. Anchar tag-Used to define ana hyperlink which is used to link from one page to another
  4. Italic tag-Defines a part of text in an alternate voice
  5. Paragraph tag-Defines a paragraph
  6. Defines a list item
  7. Bold tag-Specifies bold text without any extra importance
  8. Horizontal lines tag-Used to visually break-up sections of a document
